# Player-Earning Economy Plan

## Core Rule

Training creates power. Gameplay creates virtual value. Verified achievement and player-created value create real earning opportunities.

The core fitness layer must stay account-bound and non-purchasable:

- Power
- Vitality
- Agility
- Endurance
- Control
- Hero XP
- Battle attempts
- Consistency rewards

Money must never replace training.

## Economy Layers

1. Training progression: permanent account-bound hero growth.
2. Game economy: Gold, Essence, equipment, earned chests, crafting and upgrades.
3. Achievement economy: fixed-content verified achievement rewards that may later qualify for a controlled real-money marketplace.
4. Creator and service economy: programmes, coaching, guilds, challenge packs and other player-created value.

## Prototype Scope

The current demo should only implement Phase 1 game-economy features:

- Gold
- Essence
- Earned chest fragments and earned chests
- Equipment origin
- Item binding status
- Equip, upgrade and dismantle actions
- No cash marketplace
- No Gems
- No withdrawals
- No cash-eligible random loot

## Hard Boundaries

Gold, Essence, equipment and random earned chests cannot be withdrawn or converted to real money.

Purchased currency must never buy:

- Training attributes
- Permanent combat power
- Auction-house Gold
- Tradeable equipment
- Cash-eligible achievement items

Real-money earning should come only from:

- Sponsored challenge prizes
- Creator royalties
- Coaching income
- Guild subscription shares
- Approved fixed-content achievement sales
- Referral or partner commissions

## Roadmap

Phase 1: strengthen the core RPG with five-stat progression, clear rewards, one polished chapter, equipment, Gold, Essence, earned chests, item binding and basic crafting.

Phase 2: move ownership to authenticated cloud accounts with training, inventory and currency ledgers.

Phase 3: add a Gold auction house with escrow, fees, search and fraud alerts.

Phase 4: add direct cosmetics, Hero Pass and cosmetic-only purchases.

Phase 5: pilot sponsored challenges with no entry fee, manual verification and fixed prizes.

Phase 6: launch a creator marketplace with approved trainers and reviewed programmes.

Phase 7: add a controlled achievement marketplace for fixed-content verified rewards.

Phase 8: add guild memberships and revenue sharing.

Phase 9: expand into social/world features while keeping verified fitness and payouts on the standalone platform.
